Overview of My Cloud and DevOps Journey

 My journey into the world of Cloud Computing and DevOps has been a transformative experience, marked by hands-on projects, continuous learning, and a passion for automation. Starting with foundational concepts, I have steadily progressed to implementing real-world solutions that bridge the gap between development and operations.

I began by mastering containerization tools like Docker, where I developed a three-tier application and deployed Flask microservices in separate containers for frontend and backend services. Through this, I gained expertise in Docker images, containers, networks, volumes, and Docker Compose, solidifying my understanding of container-based architectures.

To deepen my knowledge of cloud platforms, I explored Infrastructure as Code (IaC) techniques, automating AWS resource provisioning and deployment processes. Using tools like Terraform and YAML, I successfully demonstrated end-to-end deployment pipelines, showcasing scalable, efficient, and repeatable infrastructure management.

Venturing into Kubernetes, I set up and configured clusters using MicroK8s, gaining proficiency in managing workloads, scaling applications, and optimizing resource utilization. These experiences have given me a solid foundation to tackle complex deployment scenarios in production environments.
